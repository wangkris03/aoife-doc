# 资源及跳转

## 外部资源

外部链接、外部图片和普通的 Markdown 语法保持一致

语法：

```md
![](https://t7.baidu.com/it/u=1595072465,3644073269&fm=193&f=GIF)
```

例子：

![](https://t7.baidu.com/it/u=1595072465,3644073269&fm=193&f=GIF)

## 页面跳转

页面跳转, 括号后面传入 `#/` + `文件名` 即可

语法：

```md
[AoifeDoc 简介](#/AoifeDoc简介)
```

例子： [AoifeDoc 简介](#/AoifeDoc简介)

## 站内资源引用

站内资源，使用资源文件相对于跟路径：

语法：

```md
![](/md/example.png)
```

例子： ![](/md/example.png)
