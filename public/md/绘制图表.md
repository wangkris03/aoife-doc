# 绘制图表

aoife-doc 内置了 mermaid 进行图表渲染

完整的 mermaid 语法请查阅：[https://mermaid-js.github.io/mermaid/](https://mermaid-js.github.io/mermaid/)

若您使用 ```graph 开头的代码块，aoife-doc 就会使用 mermaid 进行解析，如以下例子：

## 饼图

语法：

````md
```graph
pie
    title Key elements in Product X
    "Calcium" : 42.96
    "Potassium" : 50.05
    "Magnesium" : 10.01
    "Iron" :  5
```
````

例子：

```graph
pie
    title Key elements in Product X
    "Calcium" : 42.96
    "Potassium" : 50.05
    "Magnesium" : 10.01
    "Iron" :  5
```
